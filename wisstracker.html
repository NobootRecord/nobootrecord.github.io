Windows System Sounds Tracker (WiSSTracker)

Windows System Sounds Tracker (или же просто WiSSTracker) - очередная хрень, написанная мной на Delphi 7, а именно - программа, которая позволяет делать "музыку" из системных звуков Windows.


Данная программа (внешне) пародирует известную программу FastTracker 2. Есть русский и английский языки интерфейса, анимации плавного появления/исчезновения окна программы и множество других весьма интересных фич.
Программа написана мной чисто по приколу, но может пригодиться в экспериментальных и бытовых целях :)
