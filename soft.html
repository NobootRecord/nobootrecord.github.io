Мои программы

На этой странице перечислены мои программы, над которыми я сейчас работаю. В основном это различные системные утилиты, графические/математические приколы и просто программы-шутки.
Все мои программы абсолютно бесплатны, а некоторые из них даже распространяются с открытым исходным кодом.


String Hacker - утилита для "взлома" и модификации строковых и текстовых данных с использованием различных криптографических алгоритмов и операций.

Windows System Sounds Tracker - с этой программой можно делать "музыку" из системных звуков Windows.

Ваш компьютер сильно тормозит, а купить новый у вас нет возможности?! Не спешите его выбрасывать - ускорить его работу вам поможет наша замечательная утилита Turbo RAM Downloader!!!

Prime Numbers Finder - небольшая математическая программа, которая ищет и выводит простые числа в определённом диапазоне.

Sticky95 - аналог приложения "Заметки" из Windows 10 для Windows 9x.

Dancing Polygon - весьма странная и никому не понятная хрень, которая рисует прикольные фигурки с использованием GDI-функций.

KamstRec - утилита, которая наблюдает за состоянием мыши и клавиатуры в системе.
