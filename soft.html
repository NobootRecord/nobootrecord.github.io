Программы

  Бесплатные (и даже открытые) программы, написанные и разработанные мной.
Их можно свободно использовать и распространять (но только не для получения коммерческой прибыли). Автор не несёт ответственности за все потери данных, ущерб, повреждения и прочие неприятности, как-либо связанные с (не-)правильным использованием этих программ.

  String Hacker - утилита для "взлома", искажения и модификации строковых данных - отсюда и название сей замечательной программы.

  KamstRec - программа для получения сведений о текущем состоянии клавиатуры и мыши. Весьма странная и никому не понятная хрень.

  Ваш компьютер сильно "тупит", а купить новый у вас нет возможности?!
Не спешите его выбрасывать - ускорить его работу поможет наша замечательная утилита Turbo RAM Downloader!!!

  Dancing Polygon - ещё одна весьма странная и никому не понятная хрень, которая рисует прикольные фигурки с использованием GDI-функций.

  Sticky95 - приложение "Заметки" для Windows 9x.

  WiSSTracker - программа, с которой можно делать "музыку" из системных звуков Windows.
