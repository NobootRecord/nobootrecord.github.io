Super Internet File Downloader



Весьма неудачная попытка создать простую и понятную программу для загрузки файлов из интернетов.
Создана чисто по фану для прикола, не более.







Я создал эту программу чисто ради прикола, как графическую обёртку для известной утилиты CERTUTIL, с помощью которой можно скачивать файлы из интернетов. Поэтому на системах старше, чем Windows Vista эта программа не будет работать, даже не пытайтесь. Кстати, прога с открытым исходным кодом, мне его не жалко :)
